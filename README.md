## 构建与运行

### 环境要求
- GCC编译器
- Make工具

### 构建步骤

1. 克隆仓库到本地
```bash
git clone <repository-url>
cd p1-main
```

2. 使用Makefile构建项目
```bash
make all
```

3. 运行测试程序
```bash
./bin/test_main
```

4. 清理构建文件
```bash
make clean
```

## 如何扩展

1. 在`include/`目录下创建新的数据结构头文件
2. 在`src/`目录下创建对应的实现文件
3. 修改`tests/main.c`添加测试代码
4. 使用`make all`重新构建
